Visual Studio Add-in to translate the selected text in the Code Editor window to a variety of languages supported by the Microsoft Translator Preview service.

Refer to the DOCUMENTATION tab for How to Install, How to enable/disable instantly using Visual Studio's Add-in manager, How to Use, and How to Uninstall. And a little blog - http://renouncedthoughts.wordpress.com/2013/08/30/translatemaid-translate-text-right-inside-visual-studio-code-window/




Tested on:

Windows 7 x64, Windows Vista x86 with 
    1. Visual Studio 2012
    2. Visual Studio 2010
    3. Visual Studio 2008 (Includes BIDS)
 

Comments, enhancement requests are welcome. I would like to extend Translate.TryTranslate AddIn for Sql Server Management Studio as well; if interested to contribute, contact me at http://renouncedthoughts.wordpress.com/contact/ .
